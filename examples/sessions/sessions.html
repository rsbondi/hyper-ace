Tab to hypersearch

This example shows the use of a search across multiple edit sessions. Hit the [tab] key in the search box and the search will be performed according to the radio selection. Otherwise, the search box functions as the ace default. Selecting a result will jump to the selection in the editor for appropriate file. The source code for this example are the files loaded into the editor.
